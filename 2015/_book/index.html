使用方法

1.Github fork

到 https://github.com/isux/tonybook fork一份代码到自己的repo，然后git clone到本地，如：

git clone https://github.com/targetkiller/tonybook.git

为了便于文章管理和阶段性书刊收归，以年份为节点。

2.安装gitbook

nodejs和npm安装请看官网。

以上都安装后请继续关键的下一步，安装gitbook。

// mac和linux用户请自行sudo
npm install -g gitbook-cli

3.生成小书

先到2015目录，cd 2015/，然后安装依赖并编译：

gitbook build

搞定，安装并编译成功后即可打开_book/里面查看静态的网页！

当然，也可以使用它自带的实时刷新服务器一边写一边预览：

gitbook serve

4.畅快地使用markdown

开写！ markdown语法多爽不用怎么介绍了，具体看看这个Github的markdown规则。 尽情享受markdown带来的写作快感！
